


usualmente en las redes hay al menos tres tipos de ataques para asegurar el acceso a los atacantes pero en base a eso se pueden evitar de diferentesds maneras.

* Ataques de reconocimiento
Los ataques de reconocimiento se basan en reunir informacion del objetivo ya sea habitos o rutinas de los usuarios, hasta averiguar las versiones del software que corre en los puertos.
* Ataques de acceso
los ataques de acceso se trantan sobre explotar las vulnerabilidaes o exploits de los software o SO del usuario asi garantizando el acceso al sistema./
* Ataques DoS.
lso ataques DoS son una exagerada cantidad de peticiones a una pagina web para tirarla con el trafico generado, la diferencia entre el ataque Dos y el DDoS es que el DoS proviene de un solo punto o una sola maquina y el DDos proviene de varios puntos o maquinas.
