Se aprenderá la diferencia entre "*Cloud Computing y Virtualization*", si bien usualmente son utilizados como sinonimos, la virtualización separa el sistema operativo del hardware. AWS provee maneras de administrar redes que pueden desplegar variedad de servicios como el cliente desee, incluyen VMs, Web APPs, VS, IoT, entre otros.
